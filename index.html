Linguagem Assembly (Baixo nível)

Esse código é formado por uma sequência de 0 e 1 que seria incompreensível para humanos. Portanto, o Assembly Language vem para facilitar o entendimento. Como as instruções da Assembly Language são instruções do código de máquina, ela fica sendo considerada uma linguagem de programação de baixo nível, ou seja, que está mais voltada à simbologia da máquina do que ao olhar humano.


Liguagens de alto nível

C++

A linguagem C++ é uma extensão da linguagem C, desenvolvida por Bjarne Stroustrup nos anos 1980. Ela combina programação orientada a objetos com programação procedural e genérica, permitindo a criação de classes e objetos. C++ é conhecida por seu desempenho eficiente e controle próximo do hardware, além de oferecer uma ampla gama de bibliotecas, como a STL. É amplamente utilizada em áreas como desenvolvimento de sistemas, jogos e aplicações de alto desempenho.


Java

Java é uma linguagem de programação orientada a objetos criada pela Sun Microsystems em 1995. É projetada para ser portátil, permitindo que programas escritos nela rodem em qualquer dispositivo com uma Máquina Virtual Java (JVM). Java oferece gerenciamento automático de memória, segurança para aplicativos, e suporte a múltiplas threads, sendo amplamente utilizada no desenvolvimento de aplicativos para desktop, web e dispositivos móveis, especialmente Android.
